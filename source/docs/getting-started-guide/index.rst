# Getting Started Guide

Welcome to the Getting Started Guide! This guide provides step-by-step help for beginners to build and deploy your first FRC\ |reg| robot quickly. Whether you're completely new to robotics or just new to FRC, this guide will walk you through everything you need to know.

This guide is designed to be followed in order, from introduction through your first autonomous routine. Each section builds on the previous one, so we recommend starting at the beginning and working your way through.
